# Do Not Steal This Repository

This repository is intentionally empty.

It exists as a protest against the use of human-created work for AI training, scraping, or dataset construction without permission, attribution, or compensation.

There is no code here.
That is the message.

## Why this repository is empty

If creators cannot control whether their work is copied into training datasets,
then withholding the work becomes the protest.

This empty repository stands for:
- consent
- attribution
- compensation
- transparency
